# deep_plateau

> Companion code for the paper **"&lt;TODO: paper title&gt;"** ([arXiv:2605.xxxx](https://arxiv.org/abs/2605.xxxx))
> by Tancredi Schettini Gherardini and Marco Usula.

Physics-informed neural networks (PINNs) for the **asymptotic Plateau problem in hyperbolic space**.
Given a Jordan curve $\gamma \subset \partial_\infty \mathbb{H}^3 \cong S^2$ or a knot
$\gamma \subset \partial_\infty \mathbb{H}^4 \cong S^3$, the trained networks parameterise a minimal
disk in $\mathbb{H}^3$ / $\mathbb{H}^4$ asymptotic to $\gamma$. In the $\mathbb{H}^4$ case the
resulting immersion generically has isolated transverse double points; the code includes a Newton
refinement pipeline that locates them and computes the sign of the local Jacobian at each one
(intersection-number contributions).

## Setup

**Python 3.12 is recommended.** (On Python ≥ 3.14, the H4 PDE module disables its
`torch.compile` fast-path and falls back to the eager evaluator — much slower but
still correct.)

```bash
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

**`surfaces_in_h4/diagrams.ipynb` additionally requires a working system Tk.**
On macOS + Homebrew Python:

```bash
brew install python-tk@3.12
```

On Debian/Ubuntu: `sudo apt install python3.12-tk`.

> **Note:** the H4 training code is **CPU-only by design** — `train_PINN_Adam` and
> `refine_PINN_lbfgs` in [`surfaces_in_h4/src/training.py`](surfaces_in_h4/src/training.py)
> pin `device="cpu"` so that the hand-rolled `torch.compile` forward chain in
> [`surfaces_in_h4/src/geometry.py`](surfaces_in_h4/src/geometry.py) can run as a single
> compiled C++ kernel. A GPU port is not currently provided.

## Quick start

Inspect a pre-trained $\mathbb{H}^3$ minimal surface:

```bash
cd surfaces_in_h3
jupyter notebook DEMO_3D.ipynb
```

Train a new $\mathbb{H}^4$ minimal-disk model from scratch (boundary = stevedore knot,
takes a few minutes on CPU at default settings):

```bash
cd surfaces_in_h4
jupyter notebook train_and_save.ipynb
```

Load a pre-trained $\mathbb{H}^4$ model and locate / refine its double points:

```bash
cd surfaces_in_h4
jupyter notebook double_points.ipynb
```

## The mathematics, in brief

Working in the upper half-space model
$\mathbb{H}^{n+1} = \{(X, Y_1, \ldots, Y_n) : X > 0\}$ with metric
$ds^2 = (dX^2 + |dY|^2)/X^2$, we look for a minimal map
$u = (\rho, v) : D^2 \to \mathbb{H}^{n+1}$ with prescribed asymptotic boundary
$\gamma \subset \{X = 0\}$. Writing $g = u^* g_{\mathbb{H}}$ for the induced metric on $D^2$,
the minimal-map (tension-field) equations are

$$\Delta_g \rho + \tfrac{1}{\rho}\bigl(|dv|_g^2 - |d\rho|_g^2\bigr) = 0,
\qquad \Delta_g v_i - \tfrac{2}{\rho}\langle d\rho, dv_i\rangle_g = 0.$$

The network parameterises a trial map that already satisfies the boundary condition by
construction:

$$X = \rho_0(x,y) \cdot e^{\mathrm{NN}^X(x,y)}, \qquad
  Y = E_\gamma(x,y) + \rho_0(x,y)^{k} \cdot \mathrm{NN}^Y(x,y),$$

where $\rho_0 = (1-|xy|^2)/(1+|xy|^2)$ is a stereographic boundary-defining function
and $E_\gamma$ is a stereo-biharmonic extension of $\gamma$ built from its Fourier
coefficients. The training loss is the squared $L^2$ norm of the tension-field residual
above, evaluated on collocation points sampled inside $D^2$.
